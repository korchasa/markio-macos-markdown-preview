# Manual checklist — FR-APPEARANCE (follow system light/dark)

Reviewer: maintainer. Run `make dev` with a document containing a Mermaid
diagram and a code block, then:

- [ ] Launch in Light mode → content uses a light theme (background, code, diagram).
- [ ] Switch System Settings ▸ Appearance to Dark → preview flips to dark without restart.
- [ ] Mermaid diagram re-themes to the dark palette after the switch.
- [ ] Syntax-highlight colors switch to the dark variant.
- [ ] Native window chrome (toolbar, title) matches the system appearance.
